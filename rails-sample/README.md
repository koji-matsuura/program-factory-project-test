# Rails Sample Application

This is a test Rails application for demonstrating agentcore Phase 0-5 execution.

## Structure

- `app/models/`: User, Issue models
- `app/controllers/`: UsersController, IssuesController
- `spec/`: RSpec tests for models and controllers
- `config/routes.rb`: Route configuration

## Requirements

- Ruby 3.2.0
- Rails 7.0
- PostgreSQL

## Testing

```bash
bundle exec rspec
```
